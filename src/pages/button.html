Basic

Use the btn class on button or a elements. We currently support 3 types of buttons:

  1. Solid: This is the default and uses the primary color as background.
  2. Outline: Use btn-outline along with the btn class.
  3. Outline: Use btn-link along with the btn class.

A additional btn-danger class can be used along with any of these types if you want a red button.

                    
    <button class="btn">Solid</button>
    <button class="btn btn-outline">Outline</button>
    <button class="btn btn-link">Link</button>

    <button class="btn btn-danger">Danger</button>
    <button class="btn btn-danger btn-outline">Danger</button>
    <button class="btn btn-danger btn-link">Danger</button>

Icon Buttons

Buttons can have both icons and text.

                    
    <button class="btn">
        <i class="fas fa-heart"></i>
        Solid
    </button>
    <button class="btn btn-outline">
        <i class="fas fa-heart"></i>
        Outline
    </button>
    <button class="btn btn-link">
        <i class="fas fa-heart"></i>
        Link
    </button>

    <button class="btn">Solid</button>
    <button class="btn btn-outline">Outline</button>
    <button class="btn btn-link">Link</button>

    <button class="btn btn-danger">
        <i class="fas fa-exclamation-circle"></i>
        Danger
    </button>
    <button class="btn btn-danger btn-outline">
        <i class="fas fa-exclamation-circle"></i>
        Danger
    </button>
    <button class="btn btn-danger btn-link">
        <i class="fas fa-exclamation-circle"></i>
        Danger
    </button>

Full Width

Use an additional class btn-wide for a button that takes full width of its parent.

                    
    <button class="btn btn-wide">Solid</button>
    <button class="btn btn-wide">
        <i class="fas fa-heart"></i>
        Solid
    </button>
    <button class="btn btn-wide btn-outline">Outline</button>
    <button class="btn btn-outline">
        <i class="fas fa-heart"></i>
        Outline
    </button>
