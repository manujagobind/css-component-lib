Lists

Lists are ordered or un-ordered collection of items. Use class list on the ol or ul elements.

Un-ordered List

  • Item 1
  • Item 2
  • Item 3
  • Item 4
                    
    <ul class="list">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
    </ul>

Ordered List

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
                    
    <ol class="list">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
    </ol>

List with no bullets

Use an additional class list-no-bullets along with the list element. Optionally, a list-item-active can be used with the list item elemnt to denote the selected list item.

  • Item 1
  • Item 2
  • Item 3
  • Item 4
                    
    <ul class="list list-no-bullets">
        <li>Item 1</li>
        <li class="list-item-active">Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
    </ul>

Horizontal List

  • Item 2

  • Item 4

        <li>Item 1</li>
        <li class="list-item-active">Item 2</li>

        <li>Item 4</li>
    </ul>

Stacked List
