Alerts Component

The following CSS import is required to use the alerts component.
The FontAwesome import is required only if you plan to use icons in your alerts.

                    
    @import url("https://pleasant-ui.netlify.app/src/styles/alert.css");
    @import url("https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css");

Simple Alerts

Alerts take 80% of the parent container's width and are center aligned.
You can also add icons to the alerts if you'd like.

This is an info alert.

This is a warning alert.

This is a success alert.

This is a danger alert.

                    
    <div class="alert alert-info">
        <i class="fas fa-question-circle"></i>
        <p>This is an info alert.</p>
    </div>
    <div class="alert alert-warn">
        <i class="fas fa-exclamation-circle"></i>
        <p>This is a warning alert.</p>
    </div>
    <div class="alert alert-success">
        <i class="fas fa-check-circle"></i>
        <p>This is a success alert.</p>
    </div>
    <div class="alert alert-danger">
        <i class="fas fa-times-circle"></i>
        <p>This is a danger alert.</p>
    </div>
