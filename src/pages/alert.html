Simple Alerts

An alert is used to bring an important notice before the eyes of the user.

To create an alert, you can enclose your text message and optional icons within a div element decorated with the alert class. You can additionally use one of alert-info, alert-warn, alert-success, alert-danger classes to signify the type of alert.

Alerts take 80% of the parent container's width and are center aligned. You can also add icons to the alerts if you'd like.

This is an info alert.

This is a warning alert.

This is a success alert.

This is a danger alert.

                    
    <div class="alert alert-info">
        <i class="fas fa-question-circle"></i>
        <p>This is an info alert.</p>
    </div>
    <div class="alert alert-warn">
        <i class="fas fa-exclamation-circle"></i>
        <p>This is a warning alert.</p>
    </div>
    <div class="alert alert-success">
        <i class="fas fa-check-circle"></i>
        <p>This is a success alert.</p>
    </div>
    <div class="alert alert-danger">
        <i class="fas fa-times-circle"></i>
        <p>This is a danger alert.</p>
    </div>

Alerts With Close Button

To use dismissable alerts, provide an icon of your choice with the alert-close class within the alert element.

This is an info alert.

This is a warning alert.

This is a success alert.

This is a danger alert.

                    
    <div class="alert alert-info">
        <i class="fas fa-question-circle"></i>
        <p>This is an info alert.</p>
        <i class="fas fa-times alert-close"></i>
    </div>
    <div class="alert alert-warn">
        <i class="fas fa-exclamation-circle"></i>
        <p>This is a warning alert.</p>
        <i class="fas fa-times alert-close"></i>
    </div>
    <div class="alert alert-success">
        <i class="fas fa-check-circle"></i>
        <p>This is a success alert.</p>
        <i class="fas fa-times alert-close"></i>
    </div>
    <div class="alert alert-danger">
        <i class="fas fa-times-circle"></i>
        <p>This is a danger alert.</p>
        <i class="fas fa-times alert-close"></i>
    </div>
