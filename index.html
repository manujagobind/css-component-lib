Pleasant UI

To make use of this component library, you need use the following CSS and JS imports in your projects. Pleasant UI comes with pre-installed FontsAwesome library for icons. Check the docs here for FontAwesome on how to use icons.

              
@import url("https://pleasant-ui.netlify.app/src/styles/alert.css");
              
              
<script src="https://pleasant-ui.netlify.app/src/scripts/alert.js"></script>

PleasantUI currently supports the following components.

  1. Alerts
  2. Avatar
  3. Badge
  4. Buttons
  5. Cards
  6. Images
  7. Input
  8. Lists
  9. Navigation
  10. Text Utilities
